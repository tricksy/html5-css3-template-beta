# Portfolio Template

A clean, minimal portfolio template for designers and developers.

## Tech Stack

- Vite 6 + Tailwind CSS v4
- Vanilla ES2025+ (no frameworks)
- CSS Scroll-driven Animations
- View Transitions API
- Dark/light mode toggle (dark default)
- Noto Sans JP + Inter variable fonts

## Sections

1. **Hero** — Name, title, bio, avatar placeholder, social links
2. **About** — Bio + skills/tech stack badges
3. **Works** — Filterable project grid (All/Web/App/Design)
4. **Experience** — Vertical timeline
5. **Contact** — Form + social links
6. **Footer** — Minimal

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Customization

- Edit placeholder content in `index.html`
- Adjust theme colors in `src/style.css` (`@theme` block)
- Replace SVG avatar with your own image
- Update social links
